# WebMarCoral

Sitio web del proyecto **Mar Coral Brand**, enfocado en la visualización de un catálogo de productos.

## Tecnologías
- HTML
- CSS
- JavaScript
- Node.js (live-server)

## Estructura
- `Index.html` → Página principal
- `productos.csv` → Datos de productos
- `package.json` → Configuración del proyecto
